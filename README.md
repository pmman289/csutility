# csutility

一个基于 Minecraft NeoForge 的模组，为游戏添加了 CS 风格的 C4 爆炸物。

## Features

- 新增 C4 物品
- 手持物品按住右键可将 C4 安放到地面
- 安放后 40 秒倒计时并爆炸
- 对已安放的 C4 持续右键 10 秒可拆除
- 支持可续播、同步的 C4 倒计时音效
- 玩家在音效播放过程中进入范围仍可听到当前进度

## Environment

- Minecraft 1.21.1
- NeoForge
- Java 21+
